# Rust FFI for Nvidia Video Codec SDK 11.0.

The FFI is generated using the SDK headers that can be found in the `./vendored` directory.

## SDK update

- Download the desired SDK version from https://developer.nvidia.com/video-codec-sdk and copy
the contents of the SDK's `Interface` directory into the `./vendored` directory.
- Update the SDK version in `./vendored/README.md`.
